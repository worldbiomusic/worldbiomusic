# 설명
- `\n`: line feed 
- `\r`: carage return
- 보통 \n은 한줄 내리기 뜻이고, \r은 해당 라인의 처음으로 커서를 움직이는 명령이다

# 한줄 내리기
- Unix: \n
- Mac: \n 또는 \r
- Windows: \n\r
