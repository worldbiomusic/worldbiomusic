- 각 포트는 미리 용도가 정해져 있음 (예. 8, 10번핀은 UART 전용 포트: 칩을 만든 제조사가 칩과 IO핀을 미리 연결해놓은 것)
- GPIO핀 개수: 26개
- GPIO 입출력 신호: 3.3V (5v전용 센서 많으므로 주의)
- GPIO extension board를 raspberrypi board에 연결할 때 바깥쪽으로 연결되게 꽂
